## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| PR diff hunks → packer | Untrusted diff content is sized (bytes) only, never interpreted, during packing |
| packed set → review verdict | Verdict must reflect ONLY reviewed files; unreviewed code must never be blessed green |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-03 | Tampering (false assurance) | gate.conclude on partial coverage | mitigate | Partial coverage forces NEUTRAL (D-23) — a clean-but-incomplete review can never report green PASS over unreviewed code |
| T-07-04 | Denial of Service | unbounded prompt size on huge PR | mitigate | Whole-file packing to max_input_tokens - output_reserve keeps the prompt under the 1MB stdin guard; no-fit -> neutral skip instead of crash (D-24) |
| T-07-05 | Information Disclosure | silent under-reporting of skipped files | mitigate | Explicit "N files not reviewed" + collapsible path list (D-21) — coverage gaps are surfaced, not hidden |
| T-07-SC | Tampering | npm/pip/cargo installs | accept | No new package installs (07-RESEARCH Package Legitimacy Audit: N/A). |
