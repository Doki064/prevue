<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Antigravity install script (curl \| bash) → runner | Third-party install script executed on the runner; tampering risk. |
| LiteLLM pricing JSON (network) → vendored snapshot | Network-fetched pricing data; only the scheduled bump touches the network, gated by human PR review. |
| Antigravity API key → agy subprocess | Engine secret injected into the child env; must be conditional + sanitized. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10-17 | Tampering | Antigravity install-script (curl\|bash) tampering | mitigate | Optional `PREVUE_ANTIGRAVITY_INSTALL_SHA256` `sha256sum -c -` gate mirroring the Cursor `PREVUE_CURSOR_INSTALL_SHA256` pattern; install runs only on non-noop, engine-gated. |
| T-10-18 | Tampering | malicious pricing JSON bump | mitigate | `update-pricing.yml` only opens a PR (D-06b) — explicitly no merge/auto-approve step; the pin change is human-reviewed before it ships. Asserted by the grep gate forbidding `gh pr merge`/`--auto`. |
| T-10-19 | Elevation of Privilege | over-broad permissions on the scheduled workflow | mitigate | Explicit minimal `permissions:` block (`contents: write` + `pull-requests: write` only); SHA-pinned actions; no `secrets: inherit`. |
| T-10-20 | Information Disclosure | Antigravity secret leaks in stderr / non-TTY wrapper | mitigate | Secret passed conditionally; existing `sanitize_stderr(secret)` path in the generic adapter unchanged; ANSI-strip operates on stdout text only. |
| T-10-21 | Spoofing | Antigravity returns empty output silently (non-TTY bug) | mitigate | Pseudo-TTY `script -qec` wrapper; empty output raises `EngineFailure` (existing invoke_subprocess_text contract) rather than a false-green; the live checkpoint confirms a real sticky is posted. |
| T-10-SC | Tampering | npm/pip/cargo installs | mitigate | No new packages. `agy` is a vendor CLI installed via checksum-gated curl script; the pricing bump uses SHA-pinned GitHub actions, human-reviewed. Per 10-RESEARCH Package Legitimacy Audit: zero new pip packages. |
</threat_model>
