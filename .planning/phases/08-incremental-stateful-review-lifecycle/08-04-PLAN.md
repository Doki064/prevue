<objective>
Build the thin GraphQL transport for review-thread resolution (D-08/D-10) and
inject the compact known-issues list into the review prompt as fenced UNTRUSTED
DATA (D-07).

GraphQL-only. A thin requests-based helper isolates the GraphQL contract in one
tested module with best-effort error handling (a 403 — the unverified
contents:write scope risk — must never fail the run). The known-issues list is
the engine-side dedupe signal (D-07); because it is engine-derived text it MUST
reuse prompt.py's UNTRUSTED-DATA fencing (SECR-02). Both touch files disjoint
from Plan 03, so this runs in parallel (Wave 2). Depends on 08-01 GraphQL fixtures.
Output: src/prevue/github/graphql.py, known-issues injection in prompt.py.
</objective>
